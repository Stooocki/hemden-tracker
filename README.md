# 👔 Hemden-Tracker

PWA zum Tracken wann Hemden zuletzt gewaschen wurden.

**Live:** https://DEIN-USERNAME.github.io/hemden-tracker/

## Auf dem Handy installieren

**iPhone (Safari):** Seite öffnen → Teilen-Button → „Zum Home-Bildschirm"  
**Android (Chrome):** Seite öffnen → Menü → „App installieren"

## Lokal entwickeln

```bash
npm install
npm run dev
```

## Deployment

Automatisch via GitHub Actions bei jedem Push auf `main`.
